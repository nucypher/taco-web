# Quick Start

### Setup
Download, install, build, and test with:

```bash
git clone https://github.com/nucypher/taco-web
cd taco-web
pnpm install
```

### Basic Development Commands

Execute common tasks with:

```bash
pnpm build
pnpm test
pnpm lint
pnpm fix
```

### Documentation

Build and publish documentation with:

```bash
pnpm doc
pnpm doc:publish
```
