nucypher-ts

Communicate with NuCypher network from browser.

This is a work in progress. Do not use in a production.

Usage

Run with:

yarn install
yarn test
yarn build

Install git hooks

npx husky install

Examples

See ./test directory for usage examples.

See nucypher-ts-demo for usage example with React app.
