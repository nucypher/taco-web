Class EIP1271AuthProvider

EIP1271AuthProvider handles EIP-1271 contract-based authentication. This provider manages authentication signatures from smart contracts that implement EIP-1271's isValidSignature solidity function.

  • Returns the authentication signature for the contract.

    Since contract signatures are created externally, this method simply returns the existing signature and metadata rather than creating a new one.
