# Contribution Guide

Download, install, build, and test with:

```bash
git clone https://github.com/nucypher/nucypher-ts
cd nucypher-ts
pnpm install
```

## Development

Execute common tasks with:

```bash
pnpm build
pnpm test
pnpm lint
pnpm fix
```

## Documentation

Build and publish documentation with:

```bash
pnpm doc
pnpm doc:publish
```
