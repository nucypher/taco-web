Function decrypt

  • decrypt(provider, domain, messageKit, porterUri?, signer?, customParameters?): Promise<Uint8Array>

  • Decrypts an encrypted message.

    Parameters

    • provider: Provider

      Instance of ethers provider which is used to interact with your selected network.

    • domain: string

      Represents the logical network in which the decryption will be performed. Must match the ritualId.

    • messageKit: ThresholdMessageKit

      The kit containing the message to be decrypted

    • Optional porterUri: string

      The URI for the Porter service. If not provided, a value will be obtained from the Domain

    • Optional signer: Signer

      An optional signer for the decryption

    • Optional customParameters: Record<string, CustomContextParam>

      Optional custom parameters that may be required depending on the condition used

    Returns Promise<Uint8Array>

    Returns Promise that resolves with a decrypted message
